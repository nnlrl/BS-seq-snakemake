# BS-seq-snakemake

```
# help
./setup.py --help

# step 1: create output dir
mkdir results

# step 2: inital your config in config/settings.yaml and config/sample_sheet.csv
./setup.py --init

# Optional: remove previous output files.
./setup.py --clean

# step 3: start the pipeline.
./setup.py -s
```
